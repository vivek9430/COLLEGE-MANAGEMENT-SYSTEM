# COLLEGE-MANAGEMENT-SYSTEM
The College Management System is a desktop application developed to manage all college activities in a digital way. It helps in maintaining student records, faculty details, course information, and fee management in one place. The system allows admin users to add, update, delete, and view student and staff details easily.
Technology Used:

Front-End: VB.NET

Back-End: MySQL

Database Connectivity: ADO.NET

Platform: Visual Studio
